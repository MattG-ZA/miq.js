Affine

The 'key' for the Affine cipher consists of 2 numbers, we'll call them 'a' and 'b'.
'a' should be relatively prime to the length of the alphabet (i.e. they should have no factors in common). 'b' should be a value between 0 and 25 (i.e. the length of the alphabet, starting at 0).

When encrypting, all letters of the alphabet are first converted to numbers ('a'=0, 'b'=1, ..., 'z'=25). Example: let the key be a = 5 and b = 7. Then take each letter of the message, convert it to its relevant number (x), and plug it into the equation:

(5 * x + 7)(mod 26).

The result of this is the position of the encrypted letter in the alphabet.

Atbash

The Atbash cipher is a substitution cipher which has a predefined key, which is a reversed version of the alphabet. The message is encrypted by replacing each letter with the letter in the corresponding position of the reversed alphabet.
